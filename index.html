Sobre

D42 Ensino de Grafo é um ferramenta interativa com intuito de apresentar uma introdução a teoria de grafos, sendo uma adaptação e tradução do projeto de código aberto D3 Graph Theory.

Este projeto foi desenvolvido como conteúdo de introdução a grafos, para a disciplina de Matemática Discreta.

Perguntas

Gratuito?

Sim. O conteúdo desta ferramenta é gratuito.

Dificuldade?

O conteúdo é apresentado de maneira simples e interativa, portanto esperamos que seja de fácil absorção.

Requisitos?

Cursado ou estar cursando a disciplina de matématica Discreta

Gera certificado?

Não
